# 操作卡：四象限定位 + 三闸检验清单

> **用法**：任何一个标的，先走三闸清单拿读数，再落到四象限定格子，格子决定动作。本卡是"原则层闸门"，buffett-gates 快照是"标的层证据"。

## 四象限（确定性 × 折扣）

- **纵轴·价值确定性** = 竞争闸 + 认知闸的联合读数（护城河多硬 × 离我圈心多近）
- **横轴·价格折扣** = 情绪闸读数（市场先生报价比锚算出的价值低多少）
- 两轴独立：确定性由企业和我决定，折扣由先生一个人决定

```
                         确定性 高
                           ^
        黄金格             |          等待格
        三闸齐开, 重注      |          伟大但太贵, 趴着
        [可乐88/苹果16/高盛08]         [1999满手现金被笑]
                           |
  折扣大 -----------------+----------------- 折扣小
                           |
        烟蒂格             |          投机格
        捡起吸一口就走      |          永不入场
        [格雷厄姆/早期巴菲特]|          [无锚的游戏]
                           |
                         确定性 低
```

## 三闸检验清单（下注前逐条过，任一闸不开 = 不动）

**锚（估值有没有地基）**
- [ ] 伊索三问答得出：丛里有鸟吗？何时出来几只？无风险利率多少？
- [ ] Owner earnings 可估：(a) 报告利润 + (b) 折旧摊销 − (c) 维持性资本开支

**竞争闸（余生够不够长）**
- [ ] 十年后护城河更宽还是更窄，说得出凭什么
- [ ] 不属于"护城河需要不断重建"的快变行业
- [ ] Mayo Clinic 测试：遮住 CEO 名字还是好生意

**认知闸-域（我配不配判断）**
- [ ] 能向聪明外行讲清它十年后怎么赚钱
- [ ] 真懂 ≠ 熟悉；圈边缘按圈外处理

**认知闸-量（判断打几折）**
- [ ] 保守估算下，价格比价值低出一截；只是略低 → 不买
- [ ] 关键假设错 30% 仍不亏本金
- [ ] 错了也不是"大错"（不伤复利轨道），仓位与把握匹配

**情绪闸（我是不是那个傻瓜）**
- [ ] 买入理由是"价值高于价格"，不是"价格在涨"
- [ ] 明天跌 50% 且五年不能卖，敢加仓
- [ ] 无杠杆、无期限错配、无被迫卖出的可能

（管理层视角的参考见 [derived-institutional-imperative](derived-institutional-imperative.md)；buffett-gates 主流程暂不设管理层闸——上市公司披露充分，公开信息足以决策。）

## 格子 → 动作

| 格子 | 动作 | 退出规则 |
|---|---|---|
| 黄金格 | 重注 | 护城河变窄或严重高估才动（经营结果打分，不用股价打分） |
| 等待格 | 趴着，入观察清单 | 等先生抑郁 |
| 烟蒂格 | 小仓或放弃 | 价差兑现即走，绝不恋战 |
| 投机格 | 永不 | —— |
