# 派生：复利机器与浮存金

> **一句话**：一锚三闸是决策规则，复利机器是它跑起来的样子——现金流入、趴着等、错价出现、三闸齐开、重注、企业复利、更多现金流回。伯克希尔本身就是这台机器的实体。

## 机器的三种燃料

1. **浮存金（float）**：保险业先收保费后赔付，中间趴着的钱可以拿去投资——成本为负的杠杆（承保有利润时，别人付钱请你用他的钱）。float 一词贯穿 1986–2024 每一封信，是全语料出现最稳定的概念
2. **留存收益**：伯克希尔从不分红——检验标准（1984 信）：**每留存一块钱，须为股东创造至少一块钱市值**；做不到就该分掉。2019 信借 Edgar Lawrence Smith 补齐理论：股权的复利魔法正来自留存再投资
3. **税务递延**：不卖就不交资本利得税，递延的税等于政府给的无息贷款——"永远持有"不只是情怀，是算术

## 机器为什么不能打断

- 频繁交易 = 每次把复利底数抽走一块（税 + 摩擦 + 再判断风险）
- 分红给"能高回报再投资的企业" = 强迫复利下车
- 杠杆 = 给市场先生一个在最糟时刻拔电源的按钮

## 对个人投资者的移植

- 没有浮存金，但有**工资现金流**——定期新钱就是你的"保费流入"，让"趴着等"不至于枯竭
- 留存检验反过来用：**你留在某标的里的每一块钱，它的再投资回报配得上吗？** 配不上而你不卖，就是把钱存进平庸的复利轨道
- 机器最脆弱的零件是持有人自己——见 [gate-mr-market](gate-mr-market.md)

## 收口检验问题

1. 我的"现金流入 → 等待 → 下注"循环闭合吗？还是满仓裸奔、先生犯病时无钱可下？
2. 这家公司留存的每一块钱滚出一块以上市值了吗（1984 检验）？
3. 我最近一次卖出，是锚变了（价值/护城河），还是只是手痒？
